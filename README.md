# ⚡ Click Speed Test

A sleek browser game that measures how fast you can click.
Built as a **fully self-contained single HTML file** with no frameworks or dependencies.

---

## 🎮 How to Play

* Choose a duration: **5s, 10s, or 15s**
* Click the glowing button as fast as possible
* **Space** or **Enter** keys also count as clicks

---

## ✨ Features

**Live Stats**

* ⏱ Time Left
* 🖱 Total Clicks
* ⚡ CPS (Clicks Per Second)
* 🏆 Best Score (saved with localStorage)

**Visual Effects**

* Animated progress bar
* Click ripple and **+1 animations**
* Neon button with press animation

**Leaderboard**

* Top **5 scores** stored in browser
* Enter your name if your score qualifies
* 🥇🥈🥉 medal icons for top ranks
* Export leaderboard as **JSON**
* Clear leaderboard anytime

---


## 💾 Data Storage

The game uses **localStorage** to store:

* High score
* Top 5 leaderboard entries

All data stays **in the browser** and is never sent to a server.
