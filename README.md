# ⚡ Click Speed Test Game

A modern **Click Speed Test web game** where players try to click as fast as possible within a fixed time limit.

Built using **HTML, CSS, and JavaScript** in a single-file application.

---

## 🚀 Features

* ⚡ Fast click speed testing
* 📊 **Live CPS (Clicks Per Second) meter**
* ⏱️ **Animated timer progress bar**
* 🏆 **Top 5 leaderboard** stored using browser localStorage
* 🎮 Keyboard support (Press **Space** or **Enter** to click)
* 📱 Fully **responsive design** for desktop and mobile
* 🎨 Modern **dark UI with warm leaderboard theme**
* 💾 High score saved automatically

---

## 🕹️ How to Play

1. Click **Start** to begin the game
2. Click the **Click Me!** button as fast as possible
3. Try to get the highest number of clicks before the timer ends
4. If your score is high enough, enter your name to appear on the **Leaderboard**

---

## 📊 Game Statistics

The game displays:

* ⏳ Time remaining
* 👆 Total clicks
* ⚡ CPS (Clicks Per Second)
* 🏆 High score

CPS is calculated as:

```
CPS = Total Clicks ÷ Time Elapsed
```

---

## 🏆 Leaderboard System

* Stores **Top 5 scores**
* Saves player **name and score**
* Data stored using **localStorage**
* Automatically sorts highest scores first
